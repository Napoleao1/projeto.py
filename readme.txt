EADME - Executando o Projeto Python
Introdução
Este documento fornece instruções sobre como instalar o Python e executar o script projeto.py diretamente com um clique do botão direito.

Passo 1: Instalar o Python
Baixar o Python:

Acesse o site oficial do Python.
Clique no botão para baixar a versão mais recente do Python compatível com seu sistema operacional (Windows, macOS ou Linux).
Instalar o Python:

Execute o instalador que você baixou.
Importante: Durante a instalação, selecione a opção "Add Python to PATH". Isso garantirá que você possa executar scripts Python diretamente.
Passo 2: Verificar a Instalação do Python
Abrir o Terminal:

No Windows, pressione Win + R, digite cmd e pressione Enter para abrir o Prompt de Comando.
Verificar a instalação:

No terminal, digite o seguinte comando e pressione Enter:
bash
Copiar código
python --version
Se o Python estiver instalado corretamente, você verá a versão instalada exibida no terminal.
Passo 3: Executar o Script projeto.py
Navegar até o arquivo:

Vá até a pasta onde o arquivo projeto.py está salvo.
Clicar com o Botão Direito:

Clique com o botão direito no arquivo projeto.py.
Selecionar "Abrir com":

No menu de contexto, passe o mouse sobre a opção "Abrir com".
Se o Python estiver listado, selecione-o. Caso contrário, clique em "Escolher outro aplicativo".
Selecionar o Executável do Python:

Na janela que se abre, navegue até o diretório onde o Python está instalado. Geralmente, o caminho é semelhante a:
makefile
Copiar código
C:\Users\[Seu Nome]\AppData\Local\Programs\Python\Python312\python.exe
Selecione o executável do Python (python.exe) e marque a caixa "Sempre usar este aplicativo para abrir arquivos .py".
Clique em OK.
Executar o Script:

Agora você pode executar o script projeto.py clicando com o botão direito e escolhendo "Abrir com Python".
Passo 4: Instalar Dependências (Se Necessário)
Se o seu script utiliza bibliotecas externas (como selenium ou openpyxl), você precisará instalá-las. Para isso, siga os passos abaixo:

Abrir o Terminal novamente e digite o seguinte comando:
bash
Copiar código
pip install selenium openpyxl
Considerações Finais
Após seguir essas instruções, você estará pronto para executar o script projeto.py com facilidade. Se tiver dúvidas ou precisar de assistência, sinta-se à vontade para entrar em contato.
